# EgeUni-Website
A modern, responsive, and mobile-first redesign concept for Ege University. Built with Vanilla JS, HTML5, and CSS3, featuring dark mode and Intersection Observer animations.

🎓 Ege University - Modern Web Concept
A complete, modern, and mobile-first front-end redesign concept for Ege University . This project aims to modernize the traditional academic web presence by prioritizing user experience, accessibility, and contemporary design principles like CSS Grid layouts and smooth interactions .

🚀 About the Project
This multi-page website concept serves as a comprehensive demonstration of modern UI/UX practices applied to the education sector . It moves away from cluttered legacy layouts, offering a clean, breathable interface with a strong visual hierarchy . Built entirely with semantic HTML5, CSS3, and Vanilla JavaScript, the project features dynamic theme switching, scroll-triggered animations, and a fully responsive grid system ``.

✨ Key Features
🌓 Dynamic Theme Switching: Implemented a robust Dark/Light mode toggle that respects the user's system preferences (prefers-color-scheme) and persists their choice using localStorage ``.

🎬 Scroll-Triggered Animations: Utilized the JavaScript IntersectionObserver API to create smooth, performant fade-in and slide-up animations as elements enter the viewport ``.

📱 Advanced Responsive Layout: Designed with a mobile-first approach using CSS Grid (auto-fit, minmax) and Flexbox to ensure perfect rendering across all device sizes ``.

📑 Multi-Page Architecture: Includes fully styled and interconnected pages for Home, About, Academic, Research, and Student sections ``.

♿ Accessibility & UX: Emphasizes high-contrast color palettes (Corporate Blue #0066A1), readable typography (Inter font family), and semantic HTML structure to enhance accessibility ``.
