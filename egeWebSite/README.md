# Ege Üniversitesi Web Sitesi Tasarımı

## Proje Hakkında

Bu proje, Ege Üniversitesi için modern, erişilebilir ve kullanıcı dostu bir web sitesi tasarımıdır. HTML ve CSS kullanılarak geliştirilmiş, JavaScript ile etkileşimli özellikler eklenmiştir.

## Dosya Yapısı

```
egeWebSite/
│
├── index.html          # Ana sayfa
├── hakkimizda.html     # Hakkımızda sayfası
├── styles.css          # Ana stil dosyası
├── script.js           # JavaScript dosyası
├── logo.png           # Ege Üniversitesi logosu (eklenmesi gerekiyor)
└── README.md          # Bu dosya
```

## Özellikler

### Tasarım Özellikleri
- ✅ Modern ve profesyonel görünüm
- ✅ Responsive tasarım (mobil, tablet, masaüstü uyumlu)
- ✅ Erişilebilirlik standartlarına uygun
- ✅ Kurumsal kimliğe uygun renk paleti
- ✅ Temiz ve düzenli kod yapısı

### Teknik Özellikler
- ✅ Semantic HTML5 kullanımı
- ✅ CSS3 ile modern stillendirme
- ✅ CSS Grid ve Flexbox ile düzen
- ✅ JavaScript ile interaktif özellikler
- ✅ Smooth scroll animasyonları
- ✅ Mobil menü navigasyonu

### Sayfalar
1. **Ana Sayfa (index.html)**
   - Hero section
   - Hızlı erişim bölümü
   - Hakkımızda özeti
   - İstatistikler
   - Haberler ve duyurular
   - İletişim bilgileri

2. **Hakkımızda Sayfası (hakkimizda.html)**
   - Tarihçe
   - Misyon ve Vizyon
   - Akademik birimler
   - Değerlerimiz

## Kurulum ve Kullanım

1. Tüm dosyaları aynı klasöre yerleştirin
2. `logo.png` dosyasını Ege Üniversitesi'nin resmi logosu ile değiştirin
3. `index.html` dosyasını bir web tarayıcısında açın
